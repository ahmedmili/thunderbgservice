# Exemples

Ce dossier contient tous les exemples d'utilisation du plugin `@ahmed-mili/capacitor-thunder-bg-service`.

## Structure

### Java
- `MyCustomBackgroundTask.java` - Exemple de tâche personnalisée en arrière-plan
- `NotificationDynamicHelper.java` - Helper pour notifications dynamiques
- `EXAMPLE_TaskWithJSCommunication.java` - Tâche avec communication JS
- `EXAMPLE_NativeJavaUsage.java` - Utilisation depuis Java natif
- `EXAMPLE_ActivityUsage.java` - Utilisation dans une Activity

### TypeScript/JavaScript
- `EXAMPLE_usage.ts` - Exemples d'utilisation de base
- `EXAMPLE_BackgroundTask_usage.ts` - Utilisation des tâches en arrière-plan
- `EXAMPLE_TaskJSCommunication.ts` - Communication avec les tâches Java

### XML
- `EXAMPLE_notification_online.xml` - Exemple de layout de notification personnalisé

### Documentation
- `EXAMPLE_README.md` - Guide général
- `EXAMPLE_BackgroundTasks_README.md` - Guide des tâches en arrière-plan
- `EXAMPLE_NativeUsage_README.md` - Guide d'utilisation native Java
- `EXAMPLE_TS_JS_Tasks_README.md` - Guide de communication JS/TS

## Utilisation

Copiez les fichiers d'exemple dans votre projet et adaptez-les selon vos besoins.

**Note**: N'oubliez pas de changer le package dans les fichiers Java (remplacez `com.yourpackage` par votre package).
