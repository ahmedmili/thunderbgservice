# Code of Conduct

## Notre Engagement

Nous nous engageons à maintenir une communauté ouverte et accueillante pour tous.

## Standards

### Comportement Attendu

- Utiliser un langage accueillant et inclusif
- Respecter les différents points de vue et expériences
- Accepter gracieusement les critiques constructives
- Se concentrer sur ce qui est meilleur pour la communauté

### Comportement Inacceptable

- Commentaires discriminatoires
- Harcèlement public ou privé
- Publication d'informations privées sans permission
- Tout autre comportement inapproprié dans un contexte professionnel

## Responsabilités

Les mainteneurs sont responsables de clarifier et d'appliquer nos standards.

## Portée

Ce code de conduite s'applique à tous les espaces du projet.

## Application

Pour signaler un comportement inacceptable, contactez les mainteneurs.
